# 概要
数理・データサイエンス・AI入門情工の演習課題について統計支援を目的としたものです。分析.xlsxが主たる操作ファイルです。デフォルトで外れ値除外の上 人口あたり列が追加されており、並び替え支援を行います。
# 使用方法
## for Mac
1. データタブ
1. データファイル右側の矢印
1. Power Query エディターの起動
1. OK
1. オプション
1. プライバシー
1. チェック[複数のソースからの...]
1. 被分析クエリ
1. target = {この中に\\"", "", \\と列名を指定してください}
1. 閉じて読み込む
1. 被分析シート
1. ここからは通常通り
